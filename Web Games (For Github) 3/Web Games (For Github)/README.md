## Features

- Responsive design for all devices (mobile, tablet, desktop)
- Drag and drop functionality for word matching games
- Touch support for mobile devices
- Audio feedback for correct/incorrect answers
- Multiple game levels and categories

## Getting Started

Simply open `index.html` in a web browser to start the application.
